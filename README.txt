---TO DO LIST APPLICATION ---

link to github: https://github.com/dmorris95/to-do-list-application

OVERVIEW
This application provides a menu with a list of options and then takes the input and does the corresponding action.

1. Add task
The application creates a list of tasks based on the users input.

2. View tasks
Application provides the user with a list of the tasks and the status of them

3. Mark task as complete
Application takes users input of which task they want mark as complete
It checks if the task hsa already been marked as complete and provides feedback if the user invalid input.

4. Delete task
Takes user input and deletes the corresponding task and its status based on the input
